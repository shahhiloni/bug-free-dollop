1. What is AWS? 
- Amazon Web Services (AWS) is a cloud computing platform provided by Amazon. It allows individuals, companies, and organizations to store data, run applications, and use computing resources over the internet instead of using physical servers.

- In simple words:
AWS = Renting IT resources (servers, storage, databases, networking, etc.) from the cloud instead of buying your own hardware.

- Example:
If a company wants to run a website, instead of buying a computer server, they can host their website on AWS servers and access it from anywhere.

2. Features of AWS

1. Scalability : AWS allows you to increase or decrease resources automatically based on demand.
Example:
If your website gets more users, AWS can automatically add more servers.

2. Pay-As-You-Go Pricing : You only pay for the services you use.
Example:
If you use a server for 2 hours, you only pay for those 2 hours.

3. High Security : 
- AWS provides strong security features such as:

1. Data encryption
2. Identity and access management
3. Network security

4. High Availability : AWS data centers are located in many regions worldwide, which ensures minimal downtime and reliable service.

5. Global Infrastructure : AWS has multiple regions and availability zones across the world, allowing applications to run closer to users for better performance.

6. Wide Range of Services
- AWS offers 200+ cloud services, such as:

1. Amazon EC2 – Virtual servers
2. Amazon S3 – Cloud storage
3. Amazon RDS – Managed database service
4. AWS Lambda – Serverless computing
5. Amazon SNS – Notification service

7. Easy Integration
- AWS services can easily integrate with other tools and applications.

Example:
You can connect your website, mobile app, or database with AWS services.

8. Automation 
- AWS allows automation using tools like scripts and APIs, which helps in faster deployment and management of applications.


---------------------  AWS Architecture ----------------
- Amazon Web Services (AWS) Architecture refers to the design and structure of cloud resources used to build and run applications on AWS.
It shows how different AWS services work together to create a secure, scalable, and highly available system.

- In simple words:
AWS Architecture = How cloud services are arranged and connected to run an application.
